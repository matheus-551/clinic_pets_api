# Clinic Pets API

## Descrição

Esta é uma API desenvolvida para fins acadêmicos na disciplina **Desenvolvimento Web II**. O objetivo deste projeto é fornecer uma interface para gerenciar informações sobre pets e seus donos, permitindo operações como criação, leitura, atualização e exclusão de dados.

## Estrutura do Projeto

O projeto é organizado nas seguintes pastas:

- `config/`: Contém arquivos de configuração, como conexão com o banco de dados e configuração de logs.
- `core/`: Contém classes e funções centrais, como tratamento de erros e repositórios base.
- `logs/`: Armazena os logs gerados pela aplicação.
- `middlewares/`: Contém middlewares para tratamento de erros e logs.
- `modules/`: Contém a lógica de negócios, dividida em módulos para donos e pets.
- `routes/`: Define as rotas da API.

## Dependências

As dependências do projeto estão listadas no arquivo `package.json`. Para instalar as dependências, execute:

```bash
npm install
```

## Uso

Para iniciar a API, basta precionar a tecla `F5` no visual studio code ou executar o cmomando

```bash
npm run dev
```
